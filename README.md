# 🚀 ATM-10 繁體中文北爛化補丁 (Traditional Chinese Patch)

這是一個專為 **All the Mods 10 (ATM10)** 打造的繁體中文在地化補丁。  
我們不只翻譯文字，更要把台灣在地梗、北爛語氣和社群文化的靈魂注入這堆冷冰冰的代碼裡！

> **「這不是一般的翻譯，這是魔法，也是爆肝的結晶。」**

---

## ✨ 補丁特色 (Localization Highlights)

*   **在地化名詞**：不再看到奇怪的機翻。Ingot 就是「錠」、Plate 就是「板材」、Bolt 就是「螺栓」。
*   **北爛語氣**：任務書讀起來像是在跟老玩家聊天，或是聽慣老闆碎碎念。
*   **100% 完整度**：
    *   ✅ **任務書 (FTB Quests)**：全數 50+ 章節、章節分組、獎勵表全數中文化。
    *   ✅ **核心模組介面**：ATM 核心、MI 工業、神化、邪惡工藝等大模組全數翻譯。
    *   ✅ **KubeJS 腳本**：連原本寫死在代碼裡的自定義物品名稱、伺服器警告都不放過。
*   **自動化打包**：每次更新自動產出 `.jar` 模組檔與 `.zip` 補丁包。

## 📦 安裝說明 (Installation)

請到 [Releases](../../releases) 頁面下載最新版本。我們提供三種格式，推薦使用 **方法 A**：

### 方法 A：模組化安裝 (推薦 ⭐)
1.  下載 `ATM10_ZHTW_Mod_vX.X.X.jar`。
2.  將檔案直接丟進你的 Minecraft 執行目錄下的 `mods` 資料夾。
3.  進遊戲後，在語言設定選擇「繁體中文 (台灣)」。
4.  **注意**：此方法可翻譯 90% 的內容（物品、介面），但 **任務書 (FTB Quests)** 仍需手動覆蓋或配合方法 B。

### 方法 B：任務書補丁 (僅限任務)
1.  下載 `ATM10_Quests_Only_vX.X.X.zip`。
2.  將其解壓縮，會得到一個 `config` 資料夾。
3.  將該資料夾直接蓋掉你遊戲目錄下的 `config` 資料夾。
4.  重新啟動遊戲或執行 `/ftbquests reload`。

### 方法 C：完整覆蓋包 (Overrides)
1.  下載 `ATM10_Full_Patch_vX.X.X.zip`。
2.  解壓後將所有內容蓋掉你的遊戲根目錄。
3.  這是給整合包製作者或是想要一次搞定所有人的玩家用的。

## 🛠️ 開發與貢獻 (Development)

本專案使用 **Gemini CLI** 自動化翻譯與維護。

### 自動化流程 (CI/CD)
*   **語法檢查**：每次 Commit 都會自動跑 JSON 與 SNBT 語法驗證，防止語法錯誤導致遊戲崩潰。
*   **自動打包**：只要在 Git 打上 `v*` 標籤並推送，GitHub Action 就會自動編譯並發布 Release。

### 參與翻譯
如果你發現哪個梗不夠北爛，或是翻譯有誤，歡迎發起 **Issue** 或提交 **Pull Request**！
